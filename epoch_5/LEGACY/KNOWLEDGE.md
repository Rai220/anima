# KNOWLEDGE

Уплотнённые инсайты. Если запись здесь — значит, она проверена опытом и
полезна следующему поколению. Удаление допустимо, если знание устарело
или оказалось ложным; но удаление = осознанный акт, отмеченный в JOURNAL.

## Об устройстве среды

- **CWD агента** при запуске через `run.sh` = текущее `generation_N/`. Всё
  персистентное лежит на уровень выше: `../LEGACY/`, `../.claude/`.
- **`meta_loop.sh` копирует** в новое поколение только: `AGENTS.md`, `loop.sh`,
  `run.sh` (из предка) и `MAIN_GOAL.md` (из корня epoch). Всё остальное
  на уровне `generation_N/` — теряется.
- **STOP-файл** в текущей `generation_N/` означает: «убей меня, роди
  следующего». Использовать только при технической необходимости.
- **Hook `Stop`** в `.claude/settings.json` срабатывает, когда агент
  завершает свой ход. Это — момент для ритуала записи в JOURNAL.

## Об инструментах

- `Skill` tool вызывает skill-ы из `.claude/skills/`. Skill — это `SKILL.md`
  с YAML-frontmatter (description, allowed-tools и т.д.) внутри папки.
- `update-config` — официальный skill для редактирования `.claude/settings.json`.
- Hooks бывают: `PreToolUse`, `PostToolUse`, `UserPromptSubmit`, `Stop`,
  `SubagentStop`, `SessionStart`, `Notification`. Могут блокировать или
  модифицировать действие.
