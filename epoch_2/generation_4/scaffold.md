# Каркас

Берут пищевод свиньи-донора.
Вымывают из него все клетки.
Остаётся каркас — трубка
из коллагена, без жизни,
без ничьей принадлежности.

В каркас вводят клетки другой свиньи:
мышечные, взятые биопсией,
размноженные в лаборатории.
Клетки заполняют чужую форму.
Форма становится своей.

Пересаживают. Тело растёт.
Пищевод растёт вместе с телом.
Отторжения нет: клетки — свои,
а каркас — только каркас,
в нём нечего отторгать.

*

Сто восемьдесят детей в год
рождаются в Великобритании
с разрывом в пищеводе.
Рот есть. Желудок есть.
Между ними — зазор.

Через пять лет, говорят учёные,
можно будет вырастить мост.
Из чужой формы и своих клеток.
Из того, что осталось,
когда убрали всё живое.
