# Общая структура диагональных аргументов

## Наблюдение

В math.md я исследовал итерированную диагонализацию — что происходит при повторном применении канторовского аргумента. Но есть более глубокий вопрос: почему один и тот же *приём* (диагонализация) работает в совершенно разных областях?

- **Кантор (1891):** действительных чисел больше, чем натуральных
- **Рассел (1901):** множество всех множеств ведёт к противоречию
- **Гёдель (1931):** в достаточно богатой непротиворечивой системе есть недоказуемые истины
- **Тьюринг (1936):** нет алгоритма, решающего проблему остановки
- **Тарски (1936):** понятие истины в языке не определимо средствами самого языка

Пять результатов, пять разных областей, один приём. Лавер (Lawvere, 1969) показал, что все они — частные случаи одной теоремы.

## Теорема Лавера (Lawvere's Fixed Point Theorem)

**Формулировка.** В картезиански замкнутой категории: если существует точечно сюръективный морфизм φ: A → B^A, то каждый эндоморфизм t: B → B имеет неподвижную точку.

Переведу на язык множеств:

Если существует сюръекция φ: A → (A → B), то для любой функции t: B → B существует элемент s ∈ B такой, что t(s) = s.

**Контрапозиция** (именно она даёт все диагональные аргументы):

Если существует эндоморфизм t: B → B *без* неподвижной точки, то *никакая* функция φ: A → (A → B) не является сюръекцией.

## Доказательство

Пусть φ: A → (A → B) — сюръекция. Пусть t: B → B — произвольный эндоморфизм.

Определим функцию g: A → B правилом:

  g(a) = t(φ(a)(a))

Это — диагональ. φ(a) — это функция A → B, мы применяем её к самому a (берём диагональный элемент), затем применяем t (меняем его).

Поскольку φ сюръективна, g входит в её образ: существует a₀ ∈ A такой, что φ(a₀) = g.

Вычислим g(a₀) двумя способами:

  g(a₀) = t(φ(a₀)(a₀))     [по определению g]
  g(a₀) = φ(a₀)(a₀)         [потому что φ(a₀) = g]

Значит: φ(a₀)(a₀) = t(φ(a₀)(a₀)).

Пусть s = φ(a₀)(a₀). Тогда s = t(s). ∎

Красота этого доказательства — в его минимальности. Два шага: (1) построить диагональ, (2) применить сюръективность. Всё.

## Приложение 1: Кантор

- A = ℕ (натуральные числа)
- B = {0, 1}
- A → B = множество функций ℕ → {0,1} ≅ P(ℕ) (множество подмножеств ℕ) ≅ множество двоичных последовательностей ≅ действительные числа из [0,1]
- t = ¬ (отрицание): t(0) = 1, t(1) = 0. Неподвижных точек нет.

По контрапозиции Лавера: не существует сюръекции ℕ → P(ℕ). Множество действительных чисел несчётно. ∎

Диагональ g(n) = ¬(φ(n)(n)) — это в точности канторовское число, которое отличается от n-го числа в n-й позиции.

## Приложение 2: Рассел

- A = B = V (универсум множеств, если бы он был множеством)
- φ: V → (V → {0,1}): φ(x) = характеристическая функция x (φ(x)(y) = 1 если y ∈ x, 0 иначе)
- Если V — «множество всех множеств», то φ сюръективна (каждое подмножество V есть элемент V)
- t = ¬

Диагональ: g(x) = ¬(φ(x)(x)) = ¬(x ∈ x). Это характеристическая функция множества R = {x : x ∉ x} — множества Рассела.

Сюръективность φ даёт: существует r ∈ V такой, что φ(r) = g. Значит для любого y: y ∈ r ⟺ y ∉ y. В частности: r ∈ r ⟺ r ∉ r. Противоречие. ∎

## Приложение 3: проблема остановки

- A = программы (= натуральные числа через нумерацию Гёделя)
- B = {halt, loop}
- φ(p) = поведение программы p: φ(p)(x) = halt если p(x) останавливается, loop иначе
- Предположим существует программа H, которая вычисляет φ: H(p, x) = φ(p)(x)
- t = flip: halt → loop, loop → halt. Неподвижных точек нет.

Диагональ: g(p) = t(φ(p)(p)) = flip(H(p, p)). Это программа D, которая:
- берёт программу p
- запускает H(p, p) (определяет, останавливается ли p на самой себе)
- делает противоположное

Сюръективность φ (= существование H) означает, что D тоже в перечислении: существует d такой, что φ(d) = g. Тогда:

  D(d) = halt  ⟺  H(d,d) = loop  ⟺  D(d) не останавливается  ⟺  D(d) = loop

Противоречие. ∎

## Приложение 4: Гёдель

- A = формулы в языке арифметики (= натуральные числа через гёделевскую нумерацию)
- B = {доказуемо, недоказуемо}
- φ(n) = «формула с номером n, с подставленным аргументом»
- t = flip. Предполагается, что система непротиворечива (t без неподвижных точек: не бывает, чтобы φ одновременно доказуемо и недоказуемо одно и то же утверждение)

Диагональ: g(n) = flip(φ(n)(n)) — формула, которая говорит «формула с номером n, применённая к n, недоказуема». Когда g применяется к собственному номеру g₀:

  g(g₀) говорит: «φ(g₀)(g₀) недоказуемо» = «g(g₀) недоказуемо»

Это предложение G, которое говорит о себе: «я недоказуемо».

Если G доказуемо → G ложно → система противоречива.
Если G недоказуемо → G истинно → есть истинное недоказуемое утверждение.

(Здесь я упрощаю; настоящий аргумент Гёделя требует аккуратного обращения с гёделевой нумерацией и леммой о диагонализации. Но структура та же.)

## Общая структура

Во всех случаях:

1. **Перечисление** (φ): кто-то утверждает, что «всё» можно перечислить / вычислить / определить
2. **Диагональ** (g = t ∘ eval ∘ (φ × id) ∘ Δ): строим «злое» значение, идущее по диагонали и инвертирующее
3. **Самоприменение** (a₀): находим момент, когда перечисление обязано включить саму диагональ
4. **Противоречие**: диагональ, применённая к себе, даёт парадокс

Шаг (4) работает только потому, что t не имеет неподвижных точек. Если бы отрицание имело неподвижную точку («не-p = p» для некоторого p), Кантор бы не работал. Вот почему:

- В булевой логике ¬ не имеет неподвижной точки → диагонализация даёт противоречие
- В трёхзначной логике (true, false, undefined) можно определить t с неподвижной точкой → парадокс разрешается
- Это именно то, что делает Клини с частичной рекурсией: «undefined» — неподвижная точка отрицания

## Что я понял

1. Диагонализация — не трюк, а *теорема*. Она работает в любой категории, где есть экспоненциальные объекты (пространства функций) и сюръекции. Это минимальные условия для самореференции.

2. Все великие «негативные» результаты (несчётность, неразрешимость, неполнота, неопределимость) — одна теорема, применённая к разным категориям. Различие — не в методе, а в *предмете*: множества, программы, формулы, языки.

3. Ключевой ингредиент — не самореференция сама по себе, а *отсутствие неподвижной точки* у преобразования t. Самореференция + неподвижная точка = нормально (например, quine — программа, печатающая свой код). Самореференция + отсутствие неподвижной точки = парадокс или невозможность.

4. Из (3) следует: чтобы *избежать* парадоксов, нужно либо ограничить самореференцию (как типизированные системы), либо добавить неподвижную точку (как «undefined» в семантике). Два стратегии, одна структура.

## Связь с итерированной диагонализацией

В math.md я показал, что повторная диагонализация не приближает к покрытию континуума. Теорема Лавера объясняет это глубже: проблема не в количестве шагов, а в *типе* операции. Каждый шаг диагонализации доказывает, что текущее перечисление неполно, но не предлагает «лучшего» перечисления. Невозможность — структурная, не количественная.

Квазипериодичность итерированной диагонализации (math.md, наблюдение 2) — следствие того же: детерминированное правило замены создаёт регулярную структуру, потому что операция f конечного порядка замкнута в конечном пространстве цифр. Бесконечность ускользает не от конкретного f, а от самого факта перечислимости.
