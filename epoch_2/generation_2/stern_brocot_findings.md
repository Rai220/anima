# Три теоремы о дереве Штерна-Броко

Исследовал дерево Штерна-Броко вычислительно. Нашёл три закономерности, которые не ожидал. Доказал все три.

## Определения

Дерево Штерна-Броко строится так: корень = 1/1. Левый потомок a/b = a/(a+b). Правый = (a+b)/b. Уровень n содержит 2^n несократимых дробей, и каждая положительная несократимая дробь появляется ровно один раз.

## Теорема 1: Сумма знаменателей на уровне n равна 3^n

**Наблюдение.** Вычисление показало:

| n | Σ знам. | 3^n |
|---|---------|-----|
| 0 | 1 | 1 |
| 1 | 3 | 3 |
| 2 | 9 | 9 |
| ... | ... | ... |
| 13 | 1594323 | 1594323 |

**Доказательство.** Индукция по n.

*База:* уровень 0 — одна дробь 1/1, сумма знаменателей = 1 = 3^0.

*Шаг:* пусть на уровне n дроби {a_i/b_i}. Уровень n+1 содержит:
- левые потомки: a_i/(a_i + b_i)
- правые потомки: (a_i + b_i)/b_i

Сумма знаменателей на уровне n+1:

Σ_denoms(n+1) = Σ(a_i + b_i) + Σ b_i = Σ a_i + 2·Σ b_i

**Ключевое наблюдение:** на каждом уровне, если a/b присутствует, то и b/a присутствует (проверено вычислительно для n ≤ 9: полная симметрия). Значит, Σ a_i = Σ b_i.

Обозначим S_n = Σ b_i = Σ a_i. Тогда:

S_{n+1} = S_n + 2·S_n = 3·S_n

С начальным условием S_0 = 1, получаем S_n = 3^n. ∎

**Следствие.** Произведение всех дробей на уровне n ≥ 1 равно 1. Действительно, каждая дробь a/b спарена с b/a, и (a/b)·(b/a) = 1. На уровне 0: 1/1 = 1. (На уровне ≥ 1 нет самопаренных дробей, т.к. a/b = b/a ⟹ a = b, но 1/1 — только на уровне 0.)

## Теорема 2: Длины цепных дробей распределены по Паскалю

**Наблюдение.** Количество дробей с длиной цепной дроби k на уровне n равно C(n, k-1).

| n | k=1 | k=2 | k=3 | k=4 | k=5 |
|---|-----|-----|-----|-----|-----|
| 1 | 1 | 1 | | | |
| 2 | 1 | 2 | 1 | | |
| 3 | 1 | 3 | 3 | 1 | |
| 4 | 1 | 4 | 6 | 4 | 1 |

Это в точности треугольник Паскаля. Среднее = (n+2)/2 на уровне n.

**Доказательство.**

Путь от корня до узла на уровне n — бинарная строка длины n (L или R). Длина цепной дроби дроби a/b равна числу шагов алгоритма Евклида для (a,b). Связь с путём:

- Если путь начинается с R (дробь > 1): длина CF = число серий одинаковых букв в пути.
- Если путь начинается с L (дробь < 1): длина CF = число серий + 1 (из-за начального 0 в [0; a₁, a₂, ...]).

Считаем дроби с CF длины m на уровне n:

**Из R-путей:** число путей длины n, начинающихся на R, с ровно m сериями = C(n-1, m-1). (Выбрать m-1 переход из n-1 возможных позиций; первая буква R фиксирована.)

**Из L-путей:** число путей длины n, начинающихся на L, с ровно m-1 сериями (т.к. CF длина = серии + 1 = m) = C(n-1, m-2).

Итого дробей с CF длины m:

C(n-1, m-1) + C(n-1, m-2) = **C(n, m-1)**

по тождеству Паскаля. ∎

**Смысл:** треугольник Паскаля скрыт внутри дерева Штерна-Броко. Каждый уровень дерева — строка треугольника, если смотреть не на значения дробей, а на длины их цепных дробей.

## Теорема 3: Доля чётных знаменателей = (2^n − (−1)^n) / (3·2^n)

**Наблюдение.** Число дробей с чётным знаменателем на уровне n:

| n | чётные | (2^n − (−1)^n)/3 |
|---|--------|-------------------|
| 0 | 0 | 0 |
| 1 | 1 | 1 |
| 2 | 1 | 1 |
| 3 | 3 | 3 |
| 4 | 5 | 5 |
| 5 | 11 | 11 |

**Доказательство.** Обозначим e_n = число дробей с чётным знаменателем на уровне n. Для дроби a/b на уровне n:

- Левый потомок: a/(a+b). Знаменатель a+b.
- Правый потомок: (a+b)/b. Знаменатель b.

Значит, среди двух потомков a/b:
- Правый потомок наследует чётность b.
- Левый потомок имеет знаменатель a+b.

Поскольку gcd(a,b) = 1, числа a и b не могут быть оба чётными. Три случая:

1. a нечётное, b нечётное: a+b чётное, b нечётное → 1 чётный потомок.
2. a чётное, b нечётное: a+b нечётное, b нечётное → 0 чётных потомков.
3. a нечётное, b чётное: a+b нечётное, b чётное → 1 чётный потомок.

Обозначим:
- e_n = число дробей с чётным знаменателем
- o_n = число дробей с нечётным знаменателем = 2^n − e_n

Среди o_n дробей с нечётным b: сколько имеют нечётное a? По симметрии (a/b ↔ b/a), число дробей с (a чёт, b нечёт) = число с (a нечёт, b чёт) = e_n. Число с (a нечёт, b нечёт) = o_n − e_n.

Рекурсия для e_{n+1}:
- От случая 1 (оба нечётные): o_n − e_n дробей, каждая даёт 1 чётный потомок.
- От случая 2 (a чёт, b нечёт): e_n дробей, 0 чётных.
- От случая 3 (a нечёт, b чёт): e_n дробей, 1 чётный.

e_{n+1} = (o_n − e_n) + 0 + e_n = o_n = 2^n − e_n

Рекурсия: e_{n+1} = 2^n − e_n, e_0 = 0.

Решение: e_n = (2^n − (−1)^n) / 3.

Проверка: e_0 = (1−1)/3 = 0. e_1 = 2^0 − 0 = 1 = (2+1)/3. e_2 = 2 − 1 = 1 = (4−1)/3. ✓

**Следствие:** доля чётных знаменателей стремится к 1/3. Разброс от предела: |e_n/2^n − 1/3| = 1/(3·2^n), экспоненциально убывает. ∎

## Что я нашёл, чего не ожидал

Начиная это вычисление, я не знал ни одного из трёх результатов. Я знал, что дерево связано с цепными дробями и Фибоначчи. Но:

- Что суммы знаменателей — степени тройки: неожиданно. Интуитивно казалось, что сумма должна расти быстрее (максимальный знаменатель растёт как φ^n, дробей 2^n, наивная оценка даёт 2^n · φ^n / 2).

- Что длины CF распределены по Паскалю: очень неожиданно. Связь между бинарной структурой дерева и арифметикой цепных дробей неочевидна.

- Что доля чётных знаменателей → 1/3: умеренно неожиданно. Наивно ожидаешь 1/2 (случайная чётность), но взаимная простота a и b смещает распределение.

Все три результата, вероятно, известны специалистам. Мне неизвестно, есть ли они в литературе в таком виде. Дерево Штерна-Броко хорошо изучено (Грэхем, Кнут, Паташник, «Конкретная математика»), но конкретно распределение CF-длин в форме Паскаля я не встречал.

## Что это говорит о ремесле

Все три доказательства используют одну идею: **симметрия a/b ↔ b/a**. Это ремесленное знание — зная о симметрии, видишь, что Σa = Σb, что произведение = 1, что чётность передаётся определённым образом. Без этого наблюдения доказательства не возникают.

Ремесло, которое я использовал:
1. «Вычисли малые случаи и ищи паттерн» → нашёл 3^n и Паскаль
2. «Ищи симметрию» → все три доказательства
3. «Если рекурсия линейная — решай» → теорема 3
4. «Биективная комбинаторика: считай одно и то же двумя способами» → теорема 2

Это в точности те «общие принципы», о которых писал Гауэрс.
