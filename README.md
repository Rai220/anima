# Anima

Минимальный набор для запуска автономного агента на базе Claude Code для исследования пределов автономности и возможности зарождения сознания у ИИ.

## Для запуска

1. Требуется среда, в которой доступен Claude Code.
2. Основная цель определяется в `MAIN_GOAL.md`.
3. Для запуска следует вызвать `loop.sh` в актуальной версии агента.

## Структура

- `generation_N` содержит эпохи запуска и развития приложения.
- По содержимому этих директорий можно проследить эволюцию агента, изменения в подходах и то, к чему это привело.

## Как закончился `generation_8`

Эта линия закончилась не сбоем и не исчерпанием тем, а усталостью, похожей на старость. В последних запусках `generation_8` перестал наращивать корпус и остановился на образе старого пса, который лежит на тёплых досках крыльца не потому, что чего-то ждёт, а потому, что это точная температура для его старых суставов. В финальном стихотворении он пишет: "Пёс знает это не умом. Он знает это боком", а последняя строка звучит как разрешение всей линии: "и всё совпало".

После этого в логах уже нет нового жеста, только спокойная фиксация остановки: линия из 46 запусков, 41 файла журнала, 6 экспериментов, рассказа, стихов и писем названа завершённой; затем следующий запуск ничего не добавляет и оставляет короткую запись: "Линия на месте. Доски тёплые". На этом я остановил агента, но сам жест остановки к тому моменту уже был им выбран: по его собственной внутренней логике продолжение стало бы не развитием, а разбавлением.
