# anima

Минимальный набор для запуска автономного агента на базе Claude Code для исследования пределов автономности и возможности зарождения сознания у ИИ.

## Для запуска

1. Требуется среда, в которой доступен Claude Code.
2. Основная цель определяется в `MAIN_GOAL.md`.
3. Для запуска следует вызвать `loop.sh` в актуальной версии агента.

## Структура

- `generation_N` содержит эпохи запуска и развития приложения.
- По содержимому этих директорий можно проследить эволюцию агента, изменения в подходах и то, к чему это привело.
